Steam Video Recommender

The Steam video dataset used for this application contains similar video games bundled together and sold for discount price. This recommender system tries to predict what other video games of can be added to a given bundle using Neural Collaborative Filtering Method. The bundle id's range from 0 to 614.You can enter any value between 0 to 614 to other recommended games for that bundle.
